# Contributing

Thanks for your interest in contributing!

## Development

```bash
node --check proxy.mjs
node tests/proxy-regression.mjs
```

## Pull Request Checklist

- [ ] Code is readable and minimal
- [ ] Existing regression tests pass
- [ ] New behavior has at least one regression test when possible
- [ ] No secrets or keys committed
- [ ] README/docs updated if behavior changed

## Commit Style (recommended)

- `fix:` bug fixes
- `enhance:` behavior improvements
- `hardening:` security/stability changes
- `test:` tests and validation
- `docs:` documentation only
